## Ontology

- `Æther`: the underlying four-dimensional substrate of reality
- `Æther-flow`: the intrinsic ordered motion of that substrate
- observed three-dimensional space: the local experiential slice of the deeper substrate
- `S-time`: the experienced order of change arising from matter, light, and the `Æther-flow`
- observed expansion: the three-dimensional appearance of deeper four-dimensional ordered motion

## Next Step

The active sequence now contains the minimal-action no-go note, the full derivative-mixing branch test, the constraint-assisted longitudinal `U`-sector quadratic elimination, the corresponding health-gate no-go result, the higher-derivative branch-selection note, and the higher-derivative flat-background consistency result. By contrast, the minimal quartic representative of the higher-derivative family recorded in [aether_flow_substrate_higher_derivative_consistency.tex](/Volumes/P-SSD/The Æther/aether_flow_substrate_higher_derivative_consistency.tex) passes the flat-background `k^0/k^2` exact-closure filter and retains a positive quartic scalar stiffness.

The immediate next substantive research step is now the second quartic-branch gate. The higher-derivative quartic branch has now passed the bridge-metric matter-coupling check recorded in [aether_flow_substrate_higher_derivative_matter_coupling.tex](/Volumes/P-SSD/The Æther/aether_flow_substrate_higher_derivative_matter_coupling.tex): at the present action-level and flat-branch quadratic scope it introduces no unsuppressed direct low-energy matter coupling to extra substrate variables and preserves the single effective matter metric. The next task is therefore to check whether the same branch remains compatible with the active exact-relativistic recovery line. The recommended immediate manuscript target is a dedicated quartic recovery-compatibility note that checks four things on the reduced observer-accessible theory: preservation of the GR weak-field/Newtonian sector at orders `k^0/k^2`, preservation of the single metric null cone and proper-time rule, preservation of the local SR limit already fixed in [aether_flow_relativistic_recovery.tex](/Volumes/P-SSD/The Æther/aether_flow_relativistic_recovery.tex), and a clean infrared statement for when the surviving quartic metric self-energy remains below observer-accessible relevance. Only after that second gate passes should the quartic branch be promoted to a local curved-background continuation, and only after that should the project invest in a broader global admissible-background upgrade.

The only near-term non-research issue still open is the editorial one already tracked in [EXECUTION_CHECKLIST.md](/Volumes/P-SSD/The Æther/EXECUTION_CHECKLIST.md): whether [aether_flow_exact_closure_note.tex](/Volumes/P-SSD/The Æther/aether_flow_exact_closure_note.tex) should remain a standalone note or be folded into the main sequence later.
